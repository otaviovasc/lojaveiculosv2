# Feature Inventory

Every migrated feature needs one row before implementation. Status values:

- `Production-ready`: runtime route/UI exists, real adapters are wired, and the
  remaining setup is environment/provider configuration.
- `Runtime-ready`: route/UI exists and production adapters are wired for owned
  data, but one external provider is not enabled until credentials/endpoints are
  supplied.
- `Placeholder`: visible navigation exists, but the feature is intentionally not
  implemented yet.
- `Migration-deferred`: product surface exists, but historical V1 migration is
  intentionally skipped for now.
